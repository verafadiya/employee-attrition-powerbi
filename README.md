Goal/Challenge:

The goal of this project was to analyze employee attrition and performance patterns for an HR team managing a 10,000-person workforce. The challenge was that attrition data existed in raw spreadsheet form with no clear visibility into which departments, salary bands, or employee segments were driving turnover. The objective was to build an interactive dashboard that transforms that raw CSV into actionable HR insights.

What I did:

Connected and cleaned a 10,000-row HR dataset in Power Query, fixing data types and engineering new columns (Salary Band, Tenure Group, Attrition Flag)
Wrote DAX measures to calculate KPIs including Attrition Rate, Attrition Count, and Average Monthly Salary
Built 5 interactive visuals — a department bar chart, salary donut, scatter plot, tenure column chart, and overtime impact chart — all linked to dynamic slicers
Applied consistent formatting, descriptive titles, and a color theme to produce a publication-ready report

Outcome/Impact:

The project resulted in a fully interactive HR analytics dashboard that reveals a 14% overall attrition rate, with clear breakdowns by department, salary tier, and tenure. It helped identify that high-performing employees with low job satisfaction represent the highest-value flight risk — a finding directly visible in the scatter plot. The project strengthened my skills in Power Query data transformation, DAX calculations, and data storytelling through visual design.
